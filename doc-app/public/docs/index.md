# Ember Common UI Documentation

Welcome to the Ember Common UI documentation! This library provides a comprehensive set of reusable UI components for Ember.js applications.

## Packages

This library is organized into three main packages:

### @triptyk/ember-input

Provides composable input components for building forms:
- Text inputs, textareas, and specialized inputs
- Checkboxes and radio buttons  
- Select dropdowns and searchable selects
- File inputs and date pickers
- Toggle switches and buttons

[View ember-input documentation](./ember-input/installation)

### @triptyk/ember-input-validation

Builds on ember-input with form validation capabilities:
- Validation-enabled versions of all input components
- TpkForm component for managing form state
- Integration with changesets for data validation
- Prefab components with common validation patterns

[View ember-input-validation documentation](./ember-input-validation/installation)

### @triptyk/ember-ui

Provides higher-level UI components:
- Modal dialogs and confirm modals
- Data tables with sorting and pagination
- Stack lists for managing collections
- Actions menus and loading indicators

[View ember-ui documentation](./ember-ui/installation)

## Getting Started

1. Install the package(s) you need
2. Add the Glint template registry to your project
3. Start using components in your templates

Each package section contains detailed installation instructions and component documentation.
